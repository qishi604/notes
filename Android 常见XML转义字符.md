# Android 常见XML转义字符

```
 - “     双引号    (&#34; 或 &quot;) 
 -  ’     单引号    (&#39; 或 &apos;) 
 - &     and符号   (&#38; 或 &amp;) 
 - <     小于号    (&#60; 或 &lt;) 
 - >     大于号    (&#62; 或 &gt;) 
 - @     at符号    (&#64;)
 -  ©     版权      (&#169;或 &copy;)
-  ®     注册商标  (&#174;或 &reg;)
-    ™     商标      (&#8482;)
-    ¥     人民币    (&#165;或 &yen;)  
-    %  &#037;
```

下面的字符在 [XML]中被定义为 空白(whitespace)字符： 

```
- 空格 &#160;
- Tab &#x0009;
- 回车 &#x000D;
- 换行 &#x000A;
```

显示 `%` 方法：
- 两个百分号表示一个百分号 `%%`
- 转义字符 `\%`
- 不格式化 `<string name="title" formatted=false>%100</string>`
