# centos hexo

## install nginx

```shell
# npm install epel-release
npm install nginx
```

## start nginx

```shell
service nginx start
```


## install nodejs

```shell
yum install -y nodejs npm --enablerepo=epel
```

## install git

```shell
yum install git
```

## install hexo

```shell
npm install hexo-cli -g
```

## init blog

```shell
cd /var/www/blog
```
